=head1 Group::Git::Taggers::Node

Package that helps L<Group::Git> tag repositories containing Node files by
looking to see if they contain a I<package.json> in the root of the repository.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
